# Clima

Ejemplo de Django para manejo de servicios de clima


## Para ejecutarlo

git clone https://github.com/danielstp/Clima.git
cd Clima
python3 -m venv venv
source venv/bin/activate
pip install django
cd Clima
../manage.py runserver
